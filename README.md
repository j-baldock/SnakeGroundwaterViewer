# SnakeGroundwaterViewer
Tools to visualize groundwater indices in the upper Snake River watershed. This viewer supports the publication "Groundwater structures fish growth and production across a riverscape", authored by Jeffrey R Baldock, Robert Al-Chokhachy, and Annika Walters, published in Freshwater Biology.
